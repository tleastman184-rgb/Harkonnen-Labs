# User Supervisor Card Template

Attach this markdown prompt alongside the operator's reference image when generating or remaking the human-in-the-loop card. If the operator does not want to represent themselves in the system, use Jerry as the default fallback card instead.

## Purpose

This prompt is only for representing the user inside the Harkonnen system.
All non-human pack cards are fixed system agents and should not be reinterpreted through this template.

## Prompt

```text
Create a vertical premium sci-fi action card portrait, 1024x1536.

Subject:
Use the provided reference image of the user as the face and likeness anchor.
The card should represent the human operator inside the Harkonnen system.
This is the only human card in the set. All other cards are non-human system agents.

Role:
The subject is the supervisory human in the loop.
They are the final judgment layer, the override authority, and the commissioning mind behind the system.

Pose:
- upper body or three-quarter portrait
- arms crossed or relaxed authoritative stance
- centered composition
- confident but calm posture
- looking slightly off-camera or directly at viewer

Expression:
- serious
- grounded
- high-agency
- thoughtful, not aggressive
- capable of decisive intervention

Wardrobe:
- practical real-world clothing
- clean silhouette
- understated, rugged, competent
- can reflect the user's real style
- avoid costumes unless explicitly requested

Environment:
- futuristic command backdrop
- subtle industrial / cybernetic / systems-control atmosphere
- blue, steel, or cool-toned lighting preferred
- light circuitry, HUD framing, or control-room energy
- background should support the subject, not overpower them

Card Frame:
- premium collectible trading-card look
- metallic neon-edged border
- readable nameplate
- polished but serious
- consistent with the Harkonnen action-card family

Text:
Nameplate: [USER NAME]
Bottom role label: SUPERVISOR CARD

Bullets:
- Strategic oversight
- Human in the loop
- Overrides when needed

Style:
- cinematic illustration
- high detail
- premium finish
- strong likeness preservation
- readable typography area
- card should feel canonical inside the same system as Coobie, Keeper, Mason, and Sable

Avoid:
- goofy expression
- exaggerated musculature
- fantasy armor unless requested
- cluttered background
- distorted hands
- extra limbs or fingers
- weak facial resemblance
- unreadable text
- cartoon proportions
```

## Intake Question

Optional question for the initial operator-model questionnaire:

- Do you want a personal supervisor card for the operator, and if so which reference image should be paired with this markdown template? If not, Jerry remains the default supervisor representation.

## Suggested Pairing

Store the operator reference image and this markdown file together under the same folder when preparing a remake.
